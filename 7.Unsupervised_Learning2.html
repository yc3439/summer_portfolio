# define a function to encode any out of bound values
def confint(mean, std, n, data):
    hi = mean + 1.96*std 
    lo = mean - 1.96*std 
    if data > hi:
        return 1
    elif data < lo:
        return 1
    else:
        return 0

Autoencoder ¶

  • Theory: The backbone of Autoencoder is deep learning. It functions kind of like Recurrent Neural Network, but Autoencoder is a special type of neural network. The middle layer of the model has fewer nodes than the begginning or the end. As the information moves from the outter layers into the inner layers, only important information is being kept. After the image has been reassembled in the outerlayer, the information moves into the beginning layer and start the looop again. During the process of dimensional reduction, outliers are the byproduct of this neural network.
  • Tran/test split ratio: 60:40
  • clf1 has hidden_neurons = [25, 2, 2, 25]
  • clf2 has hidden_neurons = [25, 10, 2, 10, 25]
  • clf3 has hidden_neurons = [25, 15, 10, 2, 10, 15, 25])
  • The number of the hidden layers and the number of neutrons in a hidden layer: if there are too many hidden layers or too many neutrons, the model tends to overfit; otherwise the model tends to underfit.

IForest ¶

  • Theory: IForest would be specified with a N number of trees. Each tree then selects any number of columns and any number of rows to calculate the arithmetic mean. The arithmetic mean for each tree will be used to calculate the anomaly score in PyOD. The lastly, there will be an overall tree being constructed. The main idea is that anomalies are farther away from the rest of the branches, and they might be hanging on top of the tree. Whereas normal data points are deeper and at the bottom of the tree.
